# Решение команды Проходили мимо хакатона MoreTech

В папке **parced_data** хранятся csv файлы с новостями из различных источников

В папке **parcers** содержатся скрипты, которые выполняют парсинг материалов из источников

`analysis.ipynb` - частотный анализ текстов. В блокноте рассматривается только частотный
анализ по категориям размеченного датасета.

`parse.sh` - скрипт, раз в 18000 секунд (5 часов) автоматически обновляющий подборку новостей,
хранящуюся в `done_data.json`
